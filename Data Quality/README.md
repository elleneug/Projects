# Data Quality Checks with PyDeequ

This folder contains scripts and datasets utilized for Data Quality checks using the `pydeequ` library.

## Library Used

We are using the `pydeequ` library, an open-source tool developed by Amazon, which is built on top of Apache Spark. It is utilized for defining and verifying data quality constraints.

## Dataset Disclaimer

Please note that the datasets included in this folder are entirely fictional. They have been created solely for the purpose of demonstrating data quality checks with `pydeequ`. Any resemblance to real persons, living or dead, or actual events is purely coincidental and unintentional.

## Purpose

The primary purpose of these scripts is to showcase various data quality checks and validations that can be performed using the `pydeequ` library, ensuring the integrity and reliability of data in analytical processes.
